Акторы
1.	Студент
2.	Администратор 
Студент:
1.	Студент заходит на сайт.
2.	Открывается страница регистрации/входа.
3.	Проходит регистрацию или вход.
4.	Открывается страница вакансий.
5.	Студент ищет нужную вакансию через прямой поиск или через фильтры.
6.	Студент читает описание вакансии.
7.	Студент нажимает кнопку “Откликнуться”.
8.	На сервер приходит отклик.
Администратор
1.	Администратор заходит на сайт.
2.	Открывается страница регистрации/входа.
3.	Проходит регистрацию или вход.
4.	Открывается страница вакансий.
5.	Администратор нажимает кнопку “Создать вакансию”.
	1.	Открывается страница с созданием новой вакансии.
	2.	Администратор выставляет все параметры и нажимает кнопку “Выставить”.
	3.	База данных обновляется новым объектом.
6.	Администратор нажимает кнопку “Редактировать”.
	1.	Открывается страница выбранной вакансии.
	2.	Администратор изменяет параметры и нажимает кнопку “Сохранить”.
	3.	В базе данный обновляется объект.
7.	Администратор нажимает кнопку “Посмотреть отклики”.
	1.	Открывается страница выбранной вакансии.
	2.	Администратор видит студентов, откликнувшихся на вакансию.
	3.	Администратор нажимает кнопку “Принять” или “Отклонить”.
	4.	Система показывает сообщение “Отклик отправлен”
